Welcome to the Indy Framework.

The IndyFramework is written in PHP and aims to provide three main areas of functionality:

1. Service based developement
2. Custom HTML tags to keep the HTML sections of your project script free
3. Dollar Notation, a way of easily putting values into a markup pages without the need for PHP echo command.

You can find the project in it's <a href="https://github.com/githublemming/indyframework">GitHub repository</a>, which also includes the latest documentation.
